Visualize all pokemon, their statistics and evolutions in an organized manner. 

This project was developed using React, NextJS, React Query and TailwindCSS.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
